常見名詞 nav / 導覽列(navbar) / navigator / 搜尋列 (search bar)

有些 header 頂端列，裡面會包涵 nav，通常有 Logo 也有導覽列，有些有搜尋框，RWD手機模式有漢堡(hamburger)

佈局 #

通常會用 ul li 或是 tr td 來群組化元件，分群方便一起調位置和特效。

Logo 常見寫法 #

  1. 單純文字連結
html
<a href="#">Logo</a>

CSS 可以加 text-decoration 清除文字底線，也可加其他裝飾，像是 a:hover

css
text-decoration: none;
  1. 有圖片的連結，外面用 a 接網址，裡面用 img 顯示圖片。
html
<a class="navbar-item" href="網址">
      <img src="圖片" alt="" width="112" height="28">
</a>
css
@media screen and (max-width: 768px){
    .navbar-item
}

範例：頂端列往下捲動不消失

CSS 裡面 position 要設 sticky，如果是要置頂的話 top 設 0

html
<div class="sticky">header</div>
css
.sticky {
  position: sticky;
  background-color: #f4a;
  top:0;
}

CSS 盒子模型 box-sizing #

box-sizing 有兩個屬性可以選，分別是 content-box 和 border-box，假設我們有一個 300px 寬的 div，它的 padding 10px，border 是 15px，如果套用 content-box 的話，總寬度由左到右就是 15px + 10px + 300px + 10px + 15px = 350px，而 border-box 的總寬度就是原先設定好的300px，扣掉 padding 和 border，內容寬度只剩 250px，一個是設定寬度就是設定內容，一個是設定寬度，寬度包含了內容+padding+border。

css
box-sizing: content-box; /* 寬度只算 content */
box-sizing: border-box; /* 寬度包含 border 和裡面的東西*/
